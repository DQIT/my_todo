# MyToDo 设计规范

> 风格关键词：**简约 · 透明 · 有质感**。视觉语言贴合 Windows 11（Fluent / Acrylic / Mica），暗色为主、半透明玻璃面板、单一强调色、大量留白。

## 1. 设计原则
- **克制**：界面元素少而精，信息层级清晰，避免装饰性噪点。
- **通透**：主面板采用半透明亚克力（Acrylic）质感，背景隐约透出底层，营造"轻"与"浮"。
- **质感**：通过细腻的边框、柔和的投影、微妙的高光与圆角，营造玻璃/磨砂材质感，而非扁平死板。
- **专注**：桌面层"无感融入壁纸"，主界面"轻量不打扰"。
- **一致**：圆角、间距、动效在全局统一，遵循 8px 栅格。

## 3. 字体与排版
- **字体**：优先 `Segoe UI Variable` → `Segoe UI` → 系统无衬线；中文 `Microsoft YaHei UI` / `微软雅黑`。
- **字号阶梯**：标题 18 / 副标题 15 / 正文 14 / 次要 12 / 桌面层默认 16（可调 12–32）。
- **字重**：标题 SemiBold(600)，正文 Regular(400)，强调 Medium(500)。
- **行高**：正文 1.5；桌面层 1.4。

## 4. 形状 · 间距 · 投影
- **圆角**：窗体 12px，卡片/输入框 8px，按钮 6px，小标签 4px。
- **栅格间距**：基于 8px（4 / 8 / 12 / 16 / 24 / 32）。
- **窗体内边距**：主内容区 16–24px。
- **边框**：统一 1px hairline，颜色见配色表。
- **投影**：
  - 窗体：`0 12px 40px rgba(0,0,0,0.35)`（柔和大投影，强化悬浮感）。
  - 卡片 hover：`0 2px 8px rgba(0,0,0,0.20)`。
- **材质**：主窗使用系统亚克力/Mica（由窗口框架提供）；CSS 层叠 `backdrop-filter: blur(20px)` 兜底。

## 5. 动效（克制、顺滑）
- **时长**：120ms（hover）/ 180ms（切换）/ 240ms（面板进出）。
- **缓动**：`cubic-bezier(0.2, 0, 0, 1)`（Fluent 标准缓动）。
- **场景**：tab 切换淡入淡出 + 轻微位移；列表项增删用淡入/高度收起；复选框勾选有打钩动画；按钮 hover 背景渐变。
- 全局禁止生硬的瞬变；尊重系统"减少动态效果"无障碍设置。

## 6. 关键组件规范
- **Tab 栏（顶部三标签）**：极简文字 tab，选中项下方 2px 强调色指示条，平滑滑动。
- **事项卡片**：复选框 + 内容（主文字）+ 时间（次要，带小图标）+ 右侧悬浮显示的编辑/删除图标。过期项时间标红。
- **复选框**：圆角方框，选中填充强调色 + 白色对勾；完成项内容变灰并加删除线。
- **按钮**：主按钮（强调色实底）/ 次按钮（透明描边）/ 图标按钮（无底，hover 浅底）。
- **输入框/选择器**：透明底 + 1px 边框，聚焦时边框变强调色 + 柔光焦点环。
- **滑块（透明度）**：细轨道 + 强调色已填充段 + 圆形拖柄，实时预览。
- **空状态**：居中淡色插画/图标 + 一句引导文案。
- **删除确认**：轻量内嵌确认（卡片内就地确认）或小弹窗，强调"永久删除"。

## 7. 桌面悬浮层视觉
- **背景**：默认全透明；透明度滑块 0%→100% 调节"背景实色度"（颜色随主题，如深色 `rgba(20,20,24,x)`）。
- **文字**：始终清晰——默认浅色 `#FFFFFF` + 文字阴影 `0 1px 3px rgba(0,0,0,0.65)`（或描边），保证任意壁纸下可读；颜色/阴影可在设置调整。
- **行样式**：每行 = 可选时间（次要、较小）+ 内容（主、较大）；过期项整行用过期色。
- **解锁态**：1px 极细强调色半透明边框 + 四角/四边缩放热区；锁定态无边框、鼠标穿透。
- **干净优先**：超出高度的内容直接裁剪，无滚动条；单行过长省略号截断。

## 8. 图标
- 线性图标（stroke 1.5–2px），风格统一（如 Lucide / Fluent System Icons）。托盘图标提供常态与（可选）状态变体。
